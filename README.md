# 🤖 Bot Builder

No-code Telegram bot builder — deploy a full-featured bot just by submitting your token.

![Python](https://img.shields.io/badge/Python-3.8+-3776AB?style=flat&logo=python&logoColor=white)
![aiogram](https://img.shields.io/badge/aiogram-3.x-26A5E4?style=flat&logo=telegram&logoColor=white)
![License](https://img.shields.io/github/license/XayitovB/bot-builder?style=flat)

## Features
- Submit a bot token → instant fully working bot
- Multi-language support (Uzbek / Russian / English)
- Auto-generated profile, statistics, and support system
- Broadcast messaging and user management
- Admin analytics dashboard

## Quick Start
```bash
git clone https://github.com/XayitovB/bot-builder
cd bot-builder
cp .env.example .env
pip install -r requirements.txt
python run.py
```

## Stack
`Python 3.8+` · `aiogram 3.2` · `aiosqlite` · `Pydantic 2` · `loguru`
